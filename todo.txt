# Nekotty

ツリービューを装備したターミナルアプリ。

- Ghosttyの libghostty-vt を用いる
- 描画部分は独自に実装する。CoreGraphicsで始める。
- macOS用アプリとする
- タブを横に並べたターミナルアプリの標準的なGUIの問題を、縦に実画面の一部を並べてグループ化することで解決する

画面はこんなかんじ : ss.png

libghostty-vtは、 ../ghostty のソースでビルドして使う。


## ツリービュー


- 左ペインはターミナルのツリービューで、右が選択されてるターミナルの全体で、selected terminalとよぶ。
- ツリービューのターミナルはすべてリアルタイム更新される。
- 多い場合はスクロールできる。cmd-Tで新タブを開く。 cmd-Nで新ウインドウを開く。複数ウインドウ対応。
- ツリーでは、末尾の5行と、左から40文字だけが描画されている。 
- マウスで選択するとselected terminalが同期する。
- ツリーはprojectごとに1階層に自動分類され、折りたたみ操作もできる。
- projectとは、 ~/${project} に相当する部分で、 ~/a/b にいるなら aプロジェクト。
- sshの場合はHostで自動分類。 
- Cmd-2-1 と、2ストロークで C2の2を選択できる。ここでCというのは macOSのcommand記号のことである。
- マウスどらっぐで並べ替え可能。


## ビルド

- Xcodeを使わず makeだけで完結させる。
- make run したらビルドと実行。
